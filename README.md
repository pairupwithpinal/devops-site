# DevOps Site

This is a simple static website deployed using **Docker** and **Nginx**.  
It is part of a DevOps learning project demonstrating:

✅ Dockerfile creation  
✅ Building a Docker image  
✅ Running a container using Nginx  
✅ Publishing the image to Docker Hub  
✅ Hosting code on GitHub
